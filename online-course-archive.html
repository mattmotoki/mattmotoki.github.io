Machine Learning

Probabilistic Graphical Models 3: Learning ↑

  • Institution: Stanford University
  • Date Completed: November 15, 2017
  • Links: Course Website, Certificate of Achievement

This course addressed how a PGM can be learned from a data set of examples. The course discussed the key problems of parameter estimation in both directed and undirected models, as well as the structure learning task for directed models.

Probabilistic Graphical Models 2: Inference ↑

  • Institution: Stanford University
  • Date Completed: October 25, 2017
  • Links: Course Website, Certificate of Achievement

This course addressed how a PGM can be used to answer questions. Even though a PGM generally describes a very high dimensional distribution, its structure is designed so as to allow questions to be answered efficiently. The course presented both exact and approximate algorithms for different types of inference tasks, and discusses where each could best be applied.

Text Mining and Analytics ↑

  • Institution: University of Illinois at Urbana-Champaign
  • Date Completed: April 01, 2017
  • Links: Course Website, Certificate of Achievement

This course covered the major techniques for mining and analyzing text data. Some of the techniques that we covered are: Latent Semantic Analysis, Latent Dirichlet Allocation, and Latent Aspect Rating Analysis.

Policy Analysis using Interrupted Time Series ↑

  • Institution: UBCx
  • Date Completed: February 21, 2017
  • Links: Course Website, Certificate of Achievement

This course comprehensively introduced interrupted time series analysis and regression discontinuity designs. These study designs allow rigorous analysis of the effect of policy changes, and are especially useful where setting up an experiment is not practical or feasible.

Probabilistic Graphical Models 1: Representation ↑

  • Institution: Stanford University
  • Date Completed: February 15, 2017
  • Links: Course Website, Certificate of Achievement

This course addressed how a PGM can be learned from a data set of examples. The course discussed the key problems of parameter estimation in both directed and undirected models, as well as the structure learning task for directed models.

The Analytics Edge ↑

  • Institution: MITx
  • Date Completed: May 28, 2015
  • Links: Course Website, Certificate of Achievement, Kaggle Competition

This course was one of the first online courses that I ever took. It was taught by Dimitris Bertsimas who is well-known in the field of Operations Research. The course was primarily business analytics covering fundamental methods such as linear regression, logistic regression, CART, clustering, data visualization, and linear programming. The problem sets were done in R and Excel (linear programming solver). The final course project was to participate in a Kaggle competition; that was my first competition and it started my love for Kaggle.

Artificial Intelligence ↑

  • Institution: BerkeleyX
  • Date Completed: May 4th, 2015
  • Links: Course Website, Certificate of Achievement

This course was taught by Dan Klein and Pieter Abbeel who are both professors at UC Berkeley and are both prominent researchers in their respective fields of NLP and reinforcement learning. The course covered search, planning, constraint satisfaction problems, decision theory, Markov decision processes, and reinforcement learning. The course assignments were in Python. The course projects centered around teaching an agent to play Pac-Man.


Big Data

Google Cloud Platform Big Data and Machine Learning Fundamentals ↑

  • Institution: Google Cloud
  • Date Completed: October 05, 2017
  • Links: Course Website, Certificate of Achievement

This course was a high-level overview to the technologies available on the Google Cloud Platform. In particular, we were introduced to: CloudSQL, Cloud Dataproc, BigQuery, Cloud Datalab, Datastore, and TensorFlow.

Introduction to Big Data ↑

  • Institution: University of California San Diego
  • Date Completed: February 14, 2017
  • Links: Course Website, Certificate of Achievement

This course high-level overview of the terminology and the core concepts behind big data problems, applications, and systems. Some of the core concepts included: Sources of big data: people, organizations, and sensors; The Five V’s of Big Data: volume, velocity, variety, veracity, valence, and value; and Hadoop, YARN, HDFS, and MapReduce.

Scalable Machine Learning ↑

  • Institution: BerkeleyX
  • Date Completed: August 06, 2015
  • Links: Course Website, Certificate of Achievement

This was the second course in a series of courses that I took on Apache Spark and big data. This course introduced the underlying statistical and algorithmic principles required to develop scalable real-world machine learning pipelines. The course presented an integrated view of data processing by highlighting the various components of these pipelines, including feature extraction, supervised learning, model evaluation, and exploratory data analysis. The course assignments were done in PySpark to implement several scalable learning pipelines.

Introduction to Big Data with Apache Spark ↑

  • Institution: BerkeleyX
  • Date Completed: July 10, 2015
  • Links: Course Website, Certificate of Achievement

This was the first course in a series of courses that I took on Apache Spark and big data. This course covered advanced undergraduate-level statistics and data analysis. The course assignments used PySpark (the Python API for Spark) to manipulate data sets in parallel. In particular, we covered log mining, textual entity recognition and collaborative filtering techniques.

Tackling the Challenges of Big Data↑

  • Institution: MIT
  • Date Completed: March 17, 2015
  • Links: Course Website, Certificate of Achievement

This was the first online course I ever took. It surveyed state-of-the-art topics in Big Data including data collection (smartphones, sensors, the Web), data storage and processing (scalable relational databases, Hadoop, Spark, etc.), extracting structured data from unstructured data, systems issues (exploiting multicore, security), analytics (machine learning, data compression, efficient algorithms), visualization, and a range of applications.

Miscellaneous

Image and Video Processing: From Mars to Hollywood with a Stop at the Hospital ↑

  • Institution: Duke University
  • Date Completed: June 20, 2017
  • Links: Course Website, Certificate of Achievement

In this course, we learned about the science behind how digital images and video are made, altered, stored, and used. In particular, we covered: Image and Video Compression (JPEG and MPEG), Spatial Processing (median filtering and histogram equalization), Image Restoration, Image Segmentation (graph cuts and active contours), Image and Video Inpainting (geometric PDEs), Sparse Modeling and Compressed Sensing, and Medical Imaging.
